#### Razor

#### Project description Razor a powerful deployment application for bare-metal and virtual servers.

#### Prereqs GEMS: mongo(if using mongo) bson bson_ext rspec(if using unit tests) yaml syntax

#### Environment variables needed RAZOR_HOME = root directory for Razor RAZOR_RSPEC_WEBPATH = local webserver path for writing Rspec HTML output (for unit testing). This is used by ‘./rspec/run_all_unit_tests_html.sh’

#### Directory structure ./bin - control scripts ./conf - configuration YAML files ./images - default images directory ./install - installation bits ./lib - root library folder ./test_scripts - testing scripts ./rspec - unit tests (RSpec)
